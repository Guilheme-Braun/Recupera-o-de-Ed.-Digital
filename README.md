# Cidadania Digital e Inteligência Artificial
**Subtítulo:** O Impacto das Deepfakes e da Desinformação na Sociedade

## 🎯 Objetivo do Site
Este site foi desenvolvido como parte da atividade de Recuperação Prática do componente de Educação Digital e IA. O objetivo principal é conscientizar a comunidade escolar sobre os riscos das mídias manipuladas (deepfakes) e o compartilhamento automatizado de notícias falsas, além de promover boas práticas de segurança digital e navegação ética através de um quiz interativo.

## 🛠️ Tecnologias Utilizadas
- HTML5 (Estruturação semântica)
- CSS3 (Estilização responsiva com Flexbox e variáveis de tema)
- JavaScript Vanilla (Manipulação de DOM para Modo Escuro e Validador de Quiz)

## 🤖 Uso de Inteligência Artificial e Créditos
Para a elaboração técnica deste projeto, foi utilizada a IA gerativa para auxiliar na estruturação rápida do código base de estilização e interatividade.
- **Prompt Utilizado no Assistente de IA:** *"Gere um código de site institucional completo usando HTML5 semântico, CSS3 com Flexbox/Dark Mode e JavaScript nativo para um quiz de fake news. O tema deve ser Cidadania Digital e Deepfakes, voltado para um projeto escolar."*
